# nuxt3-practis
『Nuxt3 フロントエンド開発の教科書』の写経プロジェクト

## Env

node v20.11.0
npm 10.3.0 or pnpm 8.15
nuxi 3.10+
Nuxt 3.9.3

## cmd

- create project
```bash
$ npx nuxi init <project-name>
$ cd <project-name>
$ npm install
## or pnpm
$ pnpm install
```

- project run
```bash
$ npm run dev
## or pnpm
$ pnpm run dev
```
